# Chinese-CLIP 监控属性颜色数据搜集清单

## 目标

补充颜色-部位绑定数据，提升文搜图中上衣、裤子、头盔、背包、鞋子、电动车、挡风被等属性的召回和绑定能力。

## 搜集原则

- 优先选择主体清晰、无遮挡、目标属性颜色可见的 crop 图。
- 一张图尽量只对应一个主目标；多人图需要裁成单目标 crop，或者只标主目标。
- 尽量少收背景复杂、颜色不确定、属性被遮挡的图。
- 每张图建议标 6-10 条 query，包括单属性 query 和组合属性 query。
- 验证集不要大量自动扩充，最好保留人工真实 query。
- P1 优先完成，P2 其次，P3 专门用于解决颜色归属混淆，P4 可选。

## 每张图标注模板

- `戴{颜色}头盔的人`
- `穿{颜色}上衣的人`
- `穿{颜色}裤子的人`
- `穿{颜色}鞋子的人`
- `背{颜色}背包的人`
- `骑{颜色}电动车的人`
- `骑带{颜色}挡风被电动车的人`
- `戴{头盔颜色}头盔穿{上衣颜色}上衣的人`
- `穿{上衣颜色}上衣{裤子颜色}裤子的人`
- `穿{上衣颜色}上衣骑{电动车颜色}电动车的人`

## 清单说明

- `P1`：最高优先级，建议先搜集。
- `P2`：第二优先级，主要补衣服/裤子的少见颜色。
- `P3`：颜色-部位反向组合，专门解决属性绑定混淆。
- `P4`：行为和手持物，可选补充。
